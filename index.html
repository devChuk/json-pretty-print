JSON Pretty Print

 

JSON Pretty Print is a ridiculously simple online web app that converts unformatted JSON into pretty-printed JSON. That's it! ü§™

Usage

  1. Simply paste unformatted JSON into the box and tap "pretty print" to see JSON prettily.
  2. Or input URI-encoded JSON with a ?json= query string into the URL bar. For example:
    https://zerodevx.github.io/json-pretty-print/?json=%7B%22foo%22%3A%22bar%20bar%20bar%22%7D
    Open
  3. NEW! Or input Base64-encoded JSON with a ?json= query string into the URL bar.
    https://zerodevx.github.io/json-pretty-print/?json=eyJmb28iOiJiYXIgYmFyIGJhciJ9
    Open

Why?

OPEN SOURCE

Source code of this app is publicly available on Github. Have a suggestion, a bug report or like to contribute? Raise an issue or open a pull-request!

BLAZINGLY FAST

Under the hood, the app uses native Javascript for encoding, decoding and prettify operations. Total payload is only 6kb gzipped, served by CDN, with no dependencies or frameworks, and runs entirely client-side; so no additional requests are made, and no round-trip time incurred. What all these mean is it's fast... blazingly fast.

MOBILE FRIENDLY

App layout is designed mobile-first, which then scales up responsively to provide a delightful experience on both mobile devices and desktops.

NO ADS OR TRACKERS

Neither ads nor analytics is installed. Yup, not even a page-view.

SECURE

This page is statically hosted on Github Pages over https. No other servers are reached or pinged. The app runs entirely client-side and no data is sent to any backend whatsoever for any nefarious reasons. Don't take my word for it - go ahead, open up your browser dev-tools and take a look.
