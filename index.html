JSON Pretty Print

 



Usage

  1. Simply paste your unformatted JSON into the text box above, then tap the button;
  2. Or input your URI-encoded JSON with a ?json= query string into your URL bar. For example:
https://jsonprettyprint.zerodevx.com/?json=%7B%22foo%22%3A%22bar%20bar%20bar%22%7D

The Boring Stuff

So I was trying to find a spam/ad/tracker-free, securely hosted, Github open-sourced, super-fast, and an all in all dead simple web app that allows the input of (ugly) JSON directly into the URL bar, then simply displays the formatted result. No frills, no fuss. One that runs entirely on the device, without dependencies, without frameworks, and without sending any data to any backend for whatever.

I couldn't, so I built one in 10 minutes.

And so it goes. This page is hosted (with love) on Github Pages and is inspectable; or just view-source it right now. Total payload is only 3kb gzipped - frameworks and bundles and build steps are so passé. Really! As usual, if you're facing any problems with this, head over to the repo, raise an issue and I'll try my best to help.
