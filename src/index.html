JSON Pretty Print

 

JSON Pretty Print is a ridiculously simple online web app that converts unformatted JSON into pretty-printed JSON. That's it! ü§™

Usage

  1. Simply paste unformatted JSON into the box and tap "pretty print" to see JSON prettily;
  2. Or input URI-encoded JSON with a ?json= query string into the URL bar. For example:
https://zerodevx.github.io/json-pretty-print/?json=%7B%22foo%22%3A%22bar%20bar%20bar%22%7D
